Name

plbrowse - show a URL using the native desktop browser

Synopsis

plbrowse [options] --url=URL

plbrowse --help
plbrowse --html-help
plbrowse --version
plbrowse --release-date
plbrowse --copyright
plbrowse --license

Description

A utility program which determines the appropriate web browser to view the given URL on the current operating system and according to the current user's preferences.

Information Options

The following options provide information about plbrowse(1) and then immediately exit:

--help
Prints a brief usage message and exits.
--html-help
Opens this page using the default web browser.
--version
Prints the version number of this software and exits.
--release-date
Prints date that this software was released and exits.
--copyright
Prints the copyright notice for this software and exits.
--license

Prints the license agreement for this software and exits.

Global Options

--url=URL

Open the given location (URL) in the default web browser.
