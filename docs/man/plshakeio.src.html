Name

plshakeio - lists information about the FileIn/SFileIn and FileOut nodes in a Shake script

Synopsis

plshakeio [options] script.shk

plshakeio --help
plshakeio --html-help
plshakeio --version
plshakeio --release-date
plshakeio --copyright
plshakeio --license

Description

This utility loads a Shake script and prints a line of information for each file I/O node found.  This information is used by the ShakeSync Tool plugin to verify that the files read and written by a Shake script correspond to the Pipeline nodes associated with the script.

The format of the information printed for these Shake file I/O nodes is:

FileIn shake-node-name pipeline-file-sequence time-shift speed in-point in-mode out-point out-mode
FileOut shake-node-name pipeline-file-sequence

The file sequence is printed using the Pipeline specification directly readable by plscript(1) and the FileSeq class from Pipeline's Java API.  If missing, the file sequence will be printed as "-".  The remaining fields printed for FileIn/SFileIn nodes is identical to values for the Shake plugs of the same name.

Information Options

The following options provide information about plshakeio(1) and then immediately exit:

--help
Prints a brief usage message and exits.
--html-help
Opens this page using the default web browser.
--version
Prints the version number of this software and exits.
--release-date
Prints date that this software was released and exits.
--copyright
Prints the copyright notice for this software and exits.
--license

Prints the license agreement for this software and exits.

Global Options

--stats=level
Print stage statistics with the given level of vebosity (level). 

By default, stage statistics are disabled.

--warnings=level

Print warning messages with the given level of vebosity (level).

By default, warning messages are disabled.
