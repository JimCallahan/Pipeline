Name

realpath - prints the absolute canonical file system path

Synopsis

realpath path

realpath --help
realpath --html-help
realpath --version
realpath --release-date
realpath --copyright
realpath --license

Description

Expands all symbolic links and resolves references to '/./', '/../' and extra '/' characters in (path) and prints the fully resolved path to stdout.  The resulting path will have no symbolic links, '/./' or '/../' components.

Information Options

--help
Prints a brief usage message and exits.
--html-help
Opens this page using the default web browser.
--version
Prints the version number of this software and exits.
--release-date
Prints date that this software was released and exits.
--copyright
Prints the copyright notice for this software and exits.
--license

Prints the license agreement for this software and exits.
