Provides network message classes sent between Pipeline applications.

Package Specification

An installed Pipeline system consists of several types of client and server programs running on a large number of hosts.  This package contains the classes that are used be client and server programs for communicating with each other.

These classes are not exposed in the Pipeline API and cannot be directly accessed by customer plugins and standalone API based applications.  An encrypted JAR file is built for these classes which is seperate from the public Pipeline API.
